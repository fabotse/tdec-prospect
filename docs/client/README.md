# Documentos do cliente — TDec

Materiais de referência fornecidos pelo cliente (TDec) para a fase de preparação de entrega.

Coloque aqui:
- E-mails / briefings do cliente (ex.: requisitos de níveis de acesso, rebranding)
- Manual de marca / brand guidelines (cores, tipografia, uso do logo)
- Qualquer documento de referência que oriente as Epics 19 (Rebranding) e 20 (Níveis de Acesso)

> Logos NÃO vão aqui — vão em [`public/brand/`](../../public/brand/README.md) (assets estáticos servidos pela aplicação).
> Esta pasta é só para **documentos de referência** (não são servidos pela app).
