# Brand assets — TDec

Coloque aqui os logos da TDec. Servidos estaticamente em `/brand/<arquivo>`.

## Arquivos atuais (Epic 19 — Rebranding)

O `<BrandLogo/>` escolhe o arquivo conforme o tema ativo:

| Arquivo                 | Cor do logo | Usado no tema | Fundo                |
| ----------------------- | ----------- | ------------- | -------------------- |
| `Logo-TDec-branco.png`  | branco      | **dark**      | escuro (logo branco) |
| `Logo-TDec-preto.png`   | preto       | **light**     | claro (logo preto)   |

Opcionais (se forem fornecidos depois):

| Arquivo                    | Uso                            |
| -------------------------- | ------------------------------ |
| `favicon.ico` / `icon.png` | favicon / app icon (Story 19.3) |

## Formato
- Atuais em **PNG**. Se houver versão **SVG** (escala sem perda), prefira-a e a `<BrandLogo/>` será atualizada.
- Fundo **transparente** recomendado.
